# Spec Kit Codex Demo

This project is initialized with GitHub Spec Kit for Codex and includes a local
demo web app: **SpecFlow Control Room**.

Run locally:

```bash
npm run dev
```

Then open:

```text
http://localhost:4173
```

Codex Spec Kit skills live in `.agents/skills/` and are invoked as:

```text
$speckit-constitution
$speckit-specify
$speckit-clarify
$speckit-plan
$speckit-tasks
$speckit-analyze
$speckit-implement
```

The seeded feature artifacts for the demo live in:

```text
specs/001-specflow-control-room/
```
